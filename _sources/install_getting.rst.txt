****************************
Getting the Software
****************************

perfSONAR |release| is the latest release. Operating system packages that can be installed using popular package managemet software (e.g. `yum`, `dnf`, `apt`) are avaiable for all perfSONAR components. In some cases, Docker containers are also available. Visit the pages below for more details on how to acquire each type of distribution:

    * See :doc:`install_el`
    * See :doc:`install_debian`
    * See :doc:`install_docker`
