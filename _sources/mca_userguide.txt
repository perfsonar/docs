****************************************
User Guide
****************************************

Before you can start using Meshconfig Administrator, you have to sign up / login.  Depending on the authentication service configuration (please see configuration page for more detail), you may be prompted to confirm the ownership of the email address by following the confirmation URL in an email sent by the authentication service.

You may edit existing entities with their name listed under "Admins" list for each entities. Certain features, such as to override the MA endpoints, requires you to have super-admin privilege. Please see :doc:`mca_configure` for more details.
